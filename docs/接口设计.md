### POST /api/web/login

```json
request
{
  "username": String,
  // 用户名
  "password": String,
  // 密码
  "code": String,
  // 验证码
}
response
{
  code: (int),
  // 状态码 200-成功，201-失败
  successMsg: (String),
  // 消息内容 
  data: {
    userInfo: Object,
    // 用户主要信息

    // 用户Token
  },
  errorMsg: (String),
  // 错误信息

  （
  String),
  // 堆栈信息
}
```
